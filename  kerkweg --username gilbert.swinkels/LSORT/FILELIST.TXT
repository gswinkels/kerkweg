The following files are included:

Executables:

DUFI.EXE        Displays files to be sorted.  Used by LSMENU. See DUFI.DOC.
LSMENU.EXE      Menu driven front end for LSORT, DOS version.
LSMENU2.EXE     Menu driven front end for LSORT, OS2 version.
LSORT.EXE       LSORT Utility, Command Line interface.  See LSORT.HLP.
				Family API for DOS and OS2.
LSRTOS2.EXE     LSRT sort filter.  Runs under DOS and OS2.
LSORTX3.EXE     Part of LSMENU. DOS and OS2.
SELEFILE.EXE    File Selection Utility.  Family API for DOS and OS2
				Used by LSMENU and LSORT.  See SELEFILE.DOC
